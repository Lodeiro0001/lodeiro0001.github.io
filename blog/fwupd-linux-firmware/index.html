fwupd

Actualiza el Firmware de tus dispositivos desde Linux

10 de diciembre de 2021 Roberto Lodeiro

  • Twitter

  • Facebook

  • Reddit

  • LinkedIn

  • Pinterest

  • Email

Si tienes un equipo Dell, Lenovo Thinkpad, Intel o HP (entre otros dispositivos y marcas) esto va a interesarte. Se acabó tener que ir a Windows para actualizar el firmware de nuestra placa base y otros dispositivos.

Vamos a ver como instalar y ulitizar fwupd para actualizar el firmware desde Linux.

Instalación

Por supuesto, fwupd se encuentra por defecto en los repositorios de las distribuciones más utilizadas. En mi caso en Debian tan solo tenemos que hacer lo siguiente:

sudo apt update
sudo apt install fwupd

Uso

Tenemos disponibles los siguientes comandos:

  • Listar los dispositivos detectados por fwupd:
fwupdmgr get-devices
  • Descargar metadata desde LVFS
fwupdmgr refresh
  • Mostrar si hay actualizaciones disponibles
fwupdmgr get-updates
  • Descargar y aplicar todas las actualizaciones disponibles para el sistema
fwupdmgr update
  • Descargar y aplicar la actualización solo de un device
fwupdmgr update DEVICEID

Con estos sencillos pasos, tendremos actualizado el firmware de nuestro equipo desde Linux, siempre que el vendedor se encuentre entre los que colaboran con LVFS.

Podemos consultar en la web de LVFS (Linux Vendor Firmware Service) todas las marcas y dispositivos soportados por el proyecto.

Espero que sea útil, ¡gracias por leerme!
